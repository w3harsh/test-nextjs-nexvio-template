# Next.js + Nexvio

Template: Next.js app with Nexvio chat widget (bottom-right). Set your key and run.

1. Copy `.env.example` to `.env.local` and set `NEXT_PUBLIC_NEXVIO_ID` (from [Nexvio dashboard](https://app.nexvio.ai)).
2. `npm install` then `npm run dev`.
3. Open [http://localhost:9000](http://localhost:9000). Widget config: `src/lib/nexvio.ts`.
